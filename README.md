# OpenVideoEnhance
Brief guide

Models available: SuperSloMo, DIAN, ESRGAN, EDVR, DeOldify, BringOldPhotosBackToLife. 

Edit options in vrt/enhance.py and run it. 
If you want to use DAIN or EDVR, run `python setup.py install` in /extension
